Npm

```
npm install @seed-catalog/stepper
```

How to use

```js
import { LitElement, html, css } from 'lit-element';
import '@seed-catalog/stepper';

class MyComponent extends LitElement {
  render() {
    return html`
      <seed-stepper .index="${0}" .size="${4}"></seed-stepper>
    `;
  }
}
```
